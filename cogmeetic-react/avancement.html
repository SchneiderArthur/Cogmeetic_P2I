1. Présentation du projet

Cogmeetic est une application web de parrainage destinée aux élèves de l'ENSC. Elle vise à remplacer les formulaires Excel et échanges informels actuellement utilisés pour constituer les binômes parrain/filleul. L'application s'appuie sur un mini-jeu de type « Tu préfères… », des profils utilisateur et un algorithme de compatibilité pour suggérer les meilleurs couples 1A / 2A.

2.2 Algorithme de matching

L'algorithme fonctionne en trois étapes prioritaires : (1) couples mutuels où chacun a mis l'autre en première position de son Top 5 ; (2) meilleurs couples issus des Top 5 croisés avec un score de préférence pondéré ; (3) appairage des élèves restants par score de compatibilité décroissant. Les élèves non appariés sont également signalés dans la vue admin.

2.3 Sécurité et persistance

  • Mots de passe hashés avec bcrypt (coût 10) — aucun mot de passe en clair
  • Authentification par token JWT valable 30 jours — toutes les routes protégées
  • Données stockées dans une base SQLite persistante sur disque
  • Vérification de l'unicité du pseudo à l'inscription

2.5 Infrastructure

  • Toutes les requêtes API utilisent un utilitaire centralisé (authFetch) qui injecte automatiquement le token JWT
  • Build de production fonctionnel (npm run build)
  • Variables d'environnement via fichier .env (URL de l'API configurable)
  • Navigation mobile avec barre du bas (Jeux, Événements, Poulpage, Chat, Profil)

4. Récapitulatif

État actuel : le prototype est fonctionnel en local. L'ensemble des fonctionnalités décrites dans le cahier des charges est implémenté à l'exception du chat en temps réel. L'authentification est sécurisée, les données sont persistantes, et le build de production est opérationnel. La prochaine étape majeure est le déploiement sur un serveur pour permettre une utilisation réelle par les 160 élèves de l'ENSC en septembre.
